# 📁 Ritika Lodhi Portfolio

A clean, responsive, and performance-optimized digital portfolio built to showcase my software engineering projects, technical skill sets, and professional journey. 

This project is built from scratch with an emphasis on clean architecture, web standards, and smooth responsive design.

---

## 🚀 Key Features Built-In

* **Mobile-First & Responsive:** Configured with advanced dynamic viewport settings to ensure an optimal viewing experience across all screen sizes (mobile, tablet, desktop).
* **Performance Optimized:** Implements Google Fonts `preconnect` and `dns-prefetch` optimization strategies to minimize structural layout shifts (CLS) and accelerate font rendering.
* **SEO & Crawler Friendly:** Pre-configured with explicit crawler commands (`index, follow`) to allow fast indexing on search engines like Google.
* **Social Graph Ready:** Utilizes Open Graph metadata (`og:title`) to ensure rich-card layouts appear cleanly when links are shared across platforms like LinkedIn, WhatsApp, and Twitter.

---

## 🛠️ Tech Stack & Architecture

* **Frontend:** HTML5, CSS3, Modern JavaScript (ES6+)
* **Typography:** [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts API
* **Deployment Target:** Vercel / Netlify (Continuous Deployment linked via Git hooks)
